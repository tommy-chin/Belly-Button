Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This dashboard contains three different charts that give insight onto the bacteria found in the belly button. These charts were created using the data gathered from human volunteers that are listed under as Test Subject ID No.. By changing the Test Subject ID No., you are able to see the the differing demographics as well as the differing Top 10 Bacteria Cultures Found, Belly Button Washing Frequency, and Bacteria Cultures Per Sample.
